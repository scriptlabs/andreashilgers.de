# Project: andreashilgers.de

This is a portfolio website for Andreas Hilgers.

## Tech Stack
- **Framework:** Next.js (App Router, TypeScript)
- **Deployment:** Vercel
- **Styling:** TailwindCSS (v4)
- **UI Components:** Radix UI (Primitives)
- **Icons:** Lucide React
- **Theming:** Multi-Theme support with Dark/Light mode using `next-themes`.

## Architecture
- `src/app/`: App router routes.
- `src/components/`: Reusable React components.
- `src/context/`: Theme and other contexts.
- `src/styles/`: Global styles and CSS variables.

## Guidelines
- Priority on accessibility and performance.
- Use CSS variables for all theme-related colors.
- Maintain German as the primary language, with English support.
